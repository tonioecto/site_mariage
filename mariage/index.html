Mariage d'Hortense et Antonin

Bienvenue sur notre site, nous avons hâte de vous retrouver le 23 septembre !

Vous pourrez trouver diverses informations dans les différents onglets du site.

Pour répondre à notre invitation, il suffit de se rendre sur l'onglet "Confirmation de venue". Nous vous demandons de répondre avant le 1 août.

Informations utiles :

  • Sans voiture, il sera compliqué de se déplacer sur place (notamment pour le trajet d'environ quarante-cinq minutes entre l'église et les forges de paimpont ). Lorsque l'onglet covoiturage sera disponible, vous pourrez le consulter pour trouver une place si vous n'avez pas de voiture.
  • Les couleurs du site ne constituent pas un code couleur à suivre pour les tenues des invités.
  • Un petit pull pourrait être utile (la salle est grande et la météo incertaine en septembre)
  • Le parking de l'église n'est pas disponible le samedi matin à cause du marché de la ville. Vous trouverez ci-dessous des solutions de parking.
